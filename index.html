TwitterBootstrapMVC.com

Skills Used: c#, ASP.NET-MVC, PostgreSql, HTML, CSS, JQuery, AJAX, Web Services

This website is an N-Tier application build with ASP.NET-MVC as a server side technology. c# was used as a programming language. PostgreSql is used as a database engine.

Client side is utilizing Razor view engine. Usability and responsiveness are achieved through the use of JQuery, Ajax and Bootstrap.

Website has HTML5 canvas reporting charts (not available to general public) implemented using JQuery.

Payment options are implemented via web services from PayPal and Stripe.

PersonalityMatch.net

Skills Used: c#, ASP.NET-MVC, SignalR, PostgreSql, HTML, CSS, JQuery, AJAX, WCF

This website is an N-Tier application build with ASP.NET-MVC as a server side technology. c# was used as a programming language. Business logic was separated using repository pattern. Data access layer is implemented via WCF service. PostgreSql is used as a database engine. Messaging mechanism is implemented using SignalR.

Client side is utilizing Razor view engine. Usability and responsiveness are achieved through the use of JQuery, Ajax and Bootstrap.

Html output is speed-optimized through the use of JavaScript and CSS bundling, html compressing using GZIP, and converting images into sprites.
